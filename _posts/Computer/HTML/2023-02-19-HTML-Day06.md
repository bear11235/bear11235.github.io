# <a href="https://www.w3schools.com/html/html_styles.asp"> 7. HTML Styles </a>

 - ```style``` 속성은 element에 색, 글꼴, 크기 등의 스타일을 추가하기 위해 사용된다.

 - ```style``` 속성은 아래와 같은 문법을 갖는다: 여기서 <b>property</b>와 <b>value</b>는 각각 CSS property, CSS value이다.

    ```html
    <tagname style="property:value;">
    ```

## Examples

 <b>1. Background Color</b>

 - CSS ```background-color``` 속성은 해당 HTML element의 HTML element의 배경색을 정의한다.

 <b>2. Text Color</b>

 - CSS ```color``` 속성은 해당 HTML element의 HTML element의 배경색을 정의한다.

 <b>3. Text Fonts</b>

 - CSS ```font-family``` 속성은 해당 HTML element의 글꼴을 결정한다.

 <b>4. Text Size</b>

 - CSS ```font-size``` 속성은 해당 HTML element의 글자 크기를 결정한다.

 <b>5. Text Alignment</b>

 - CSS ```text-alignment``` 속성은 해당 HTML element의 수평적 정렬 방법을 결정한다.

# <a href="https://www.w3schools.com/html/html_formatting.asp"> 8. HTML Text Formatting</a>

 아래는 다양한 text formatting element이다:
 
  - ```<b>```: Bold text

  - ```<strong>```: Important text. 대게 ```<b>```와 같다.

  - ```<i>```: Italic text

  - ```<em>```: Emphasized text. 대게 ```<i>```와 같다.
  
  - ```<mark>```: Marked text
  
  - ```<small>```: smaller text
  
  - ```<del>```: Deleted text. 취소선.
  
  - ```<ins>```: Inserted text. 밑줄.
  
  - ```<sub>```: 밑첨자
  
  - ```<sup>```: 위첨자
